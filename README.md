## 特色    
1.完全融入微信8.0UI风格，沉浸式体验   
2.完全免费的全功能插件  
3.非越狱可后台抢红包   
4.安全高效源码公开   
5.设置方便，手机无需越狱   
## 功能   

* [x] 自动抢红包  
* [x] 后台抢红包 
* [x] 群聊过滤
* [x] 关键词过滤
* [x] 消息防撤回     
* [x] 微信步数修改   
* [x] 小游戏作弊（剪刀石头布和掷骰子）     
* [x] CallKit锁屏接视频语音
* [x] 集赞助手   
* [x] 好友关系检测  
* [x] 朋友圈转发  
* [x] 动态聊天背景  
* [x] 动态启动图  

安装后在微信设置中点击“微信小助手”
